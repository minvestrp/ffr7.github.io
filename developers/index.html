Quickstart

Get from zero to a live bet in under 10 minutes. Base URL: https://smartsec.20.194.121.245.nip.io/api/v1

1. Get your API key

curl
curl -X POST https://smartsec.20.194.121.245.nip.io/api/v1/auth/signup \
  -H "Content-Type: application/json" \
  -d '{"email":"you@casino.com","password":"pass123","username":"myop","company_name":"My Casino"}'
{"access_token": "eyJ...", "api_key": "ss_xxxxxxxx...", "operator_id": "uuid-...", "username": "myop"}

2. Place your first bet

curl
curl -X POST https://smartsec.20.194.121.245.nip.io/api/v1/games/bet \
  -H "Content-Type: application/json" \
  -d '{"player_id":"player_1","game_type":"crash","bet_amount":10.0,"operator_api_key":"ss_xxxx..."}'
{"round_id": "uuid-...", "outcome": "win", "win_amount": 23.5, "seed_hash": "0xabc..."}

Node.js SDK (quick install)

javascript
// npm install node-fetch  (or use built-in fetch in Node 18+)
const API = 'https://smartsec.20.194.121.245.nip.io/api/v1';

async function placeBet(apiKey, playerId, gameType, amount) {
  const r = await fetch(`${API}/games/bet`, {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
      player_id: playerId,
      game_type: gameType,
      bet_amount: amount,
      operator_api_key: apiKey
    })
  });
  return r.json();
}

// Usage:
const result = await placeBet('ss_your_key', 'player_1', 'crash', 10.0);
console.log(result.outcome, result.win_amount);

Python

python
import requests

API = 'https://smartsec.20.194.121.245.nip.io/api/v1'

def place_bet(api_key, player_id, game_type, amount):
    r = requests.post(f'{API}/games/bet', json={
        'player_id': player_id,
        'game_type': game_type,
        'bet_amount': amount,
        'operator_api_key': api_key
    })
    return r.json()

result = place_bet('ss_your_key', 'player_1', 'crash', 10.0)
print(result)

Authentication

SmartSec uses JWT Bearer tokens for operator endpoints and API keys for game endpoints.

MethodEndpointDescription
POST/auth/signupRegister + auto-create operator + get API key
POST/auth/loginLogin, returns JWT token
POST/auth/demo-loginInstant demo access, no credentials needed

Using the JWT token

curl
curl https://smartsec.20.194.121.245.nip.io/api/v1/operator/overview \
  -H "Authorization: Bearer <your_token>"

Games API

8 provably fair games. Each bet returns a seed hash for on-chain verification.

MethodEndpointDescription
POST/games/betPlace a bet. Returns outcome + win amount + seed hash
POST/games/resultGet result for a round ID

Supported game_type values

  • crash — CrashX multiplier game
  • roulette — European roulette
  • blackjack — Classic blackjack
  • slots — 5-reel slots
  • dice — Over/under dice
  • mines — Mines field game
  • plinko — Plinko ball drop

Security API

Static analysis of Solidity smart contracts. Detects 10 vulnerability patterns with line numbers.

MethodEndpointDescription
POST/contracts/analyzeAudit Solidity source code. Returns risk score + vulnerabilities
GET/contracts/{id}/reportRetrieve previous analysis report

Detected vulnerability patterns

  • Reentrancy (CRITICAL) — external call before state update
  • Selfdestruct (CRITICAL) — unprotected self-destruct
  • Delegatecall (CRITICAL) — unsafe delegatecall
  • tx.origin auth (HIGH) — phishing vulnerability
  • Unchecked call (HIGH) — return value not checked
  • Integer overflow (HIGH) — Solidity <0.8.0
  • Timestamp dependence (MEDIUM) — miner manipulation
  • Inline assembly (MEDIUM) — bypasses safety checks
  • Block number (LOW) — weak randomness
  • Floating pragma (LOW) — multiple compiler versions

WebSockets

Real-time data streams over WebSocket.

javascript
// Real-time crash game stream
const ws = new WebSocket('wss://smartsec.20.194.121.245.nip.io/ws/crash');

ws.onmessage = (e) => {
  const data = JSON.parse(e.data);
  console.log(data); // { type: 'ack', ts: 1234567890 }
};

// Platform stats stream (updates every 5 seconds)
const statsWs = new WebSocket('wss://smartsec.20.194.121.245.nip.io/ws/stats');
statsWs.onmessage = (e) => {
  const { active_players, rounds_today, wagered_today } = JSON.parse(e.data);
};

Webhooks

Receive real-time notifications when bets are settled, fraud is detected, or settlements are ready.

Webhook payload example

{"event": "bet.settled", "round_id": "uuid-...", "player_id": "player_1", "outcome": "win", "win_amount": 23.5, "ts": 1714000000}

Configure your webhook URL in Operator Portal → Settings → API.
